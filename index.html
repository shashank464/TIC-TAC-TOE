TIC TAC TOE

HOW TO PLAY

  • This is a game for two players.
  • Each player gets one symbol ('X' or 'O').
  • It is a board of 9 boxes.
  • Players take turns to place their assigned symbol ('X' or 'O').
  • The player who places their symbol in any of the following patterns wins:
    • [0, 1, 2] - Top row
    • [3, 4, 5] - Middle row
    • [6, 7, 8] - Bottom row
    • [0, 3, 6] - Left column
    • [1, 4, 7] - Middle column
    • [2, 5, 8] - Right column
    • [0, 4, 8] - Left diagonal
    • [2, 4, 6] - Right diagonal
  • If no player forms any of the above patterns, the match is a draw.
